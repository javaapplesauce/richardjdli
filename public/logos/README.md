# Institution Logos

Place institution logo files here:

- `microsoft.png` - Microsoft logo
- `columbia.png` - Columbia University logo
- `caltech.png` - Caltech logo
- `uw-medicine.png` - University of Washington Department of Medicine logo

Recommended specs:
- Format: PNG or SVG with transparent background
- Size: 200x200px minimum
- Style: Official institution logos

You can download these from:
- Microsoft: https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks
- Columbia: https://visualidentity.columbia.edu/
- Caltech: https://identity.caltech.edu/
- UW Medicine: https://www.washington.edu/brand/
